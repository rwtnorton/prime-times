# prime-times

A Clojure library to generate a multiplication table of N (default 10) primes,
written to stdout.

## Usage

```
  $ lein run
  ### or via bash script:
  $ ./bin/prime-times
```

## Tests

Run tests via:
```
  $ lein test
```
